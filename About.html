CodeTogetherNG is a project that connects programers who want to learn how to develop applications in teams.

The project consists five applications

  • Main Application. It is a web application written in following technologies: C#, ASP.NET MVC, .NET Core 2.1, Nunit, FakeItEasy(tool to fake/stab/mock dependencies), Dapper(micro ORM), Bootstrap, HTML, CSS.
    The repository is located on GitHub CodeTogetherNG
  • E2E Tests. Tests writen in C#, Selenium WebDriver, Nunit, TSQL, Chromium Driver, PageObject Pattern.
    The repository is located on GitHub CodeTogetherNG_Tests
  • SQL Server DataBase Project. It is used to track data base schema changes. It uses TSQL and DacPac.
    The repository is located on GitHub CodeTogetherNG_DB
  • WebAPI. In this project I use Entity Framework Core, ASP.NET WebAPI Core.
    The repository is located on GitHub CodeTogetherNG_WebAPI
  • WebAPI Tests. HTTP Client.
    The repository is located on GitHub CodeTogetherNG_WebAPI_Tests

Azure DevOps is used as a tool to manage the project. It contains a kanban task board and pipelines: Azure DevOps
